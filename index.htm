jQuery YellowBox Plugin Beta - Examples

Soon more documentation on yellowbox other functions.

IE9+ , FF3.5+ , Google Chrome, Opera, Safari 5

Basic HTML structure

  ....
  
  <div class="yellowbox"> 
     <div class="yellowbox-title">Title Text</div>
     <div class="yellowbox-body">Any text or Messange here.</div>
  </div>  
  ....
  
  <script type="text/javascript" charset="utf-8">
    $(function(){
      $('.yellowbox').yellowBox();
    });
  </script>

Example 2 - Trigger programatic alert.

A message will appear here programatically! Click Here!!

Code:

  $('#yellowbox-2').data('api').alert({ 
    title: 'Hey There',
    body:  'This message will close in 5 seconds'
  }).closeIn(5000);


Example 3 - Trigger programatic confirm.

Hello Word!
What if it became a question with buttons and everything.

Create a question Click Here!!

Code:

  $('#yellowbox-3').data('api').confirm({
    title: '¡Question!',
    body: 'Is this process may take a few minutes, is sure to continue?',
    buttons:{
      'Aceptar':{
        onClick: function(){
          alert("Ok");
        }
      },
      'Cancelar':{
        onClick: function(){
          this.close();
        }
      }
    }
  }).enfasis(2);
